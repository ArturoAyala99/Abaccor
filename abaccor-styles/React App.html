Color de Fondo

Para agregar un color de fondo a un elemento, solo basta con agregar el nombre de la clase del color a la etiqueta y para alternar entre tonalidades agregar lighten / darken.

Este es un panel con color de fondo aplicado.

                        <div className="card-panel teal lighten-1">
                           Este es un panel con color de fondo aplicado.
                        </div>
Color de Textos

Para aplicar un color al texto, solomanete agregar al nombre de la clase de color el sufijo -text y para alternar entre tonalidades agrega el prefijo text-.

Este es un panel con color de fondo y texto aplicado.

                        <div className="card-panel teal lighten-1 blue-text text-darken-5">
                           Este es un panel con color de fondo y texto aplicado.
                        </div>
